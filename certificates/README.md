# מחולל תעודות — "סיפור חדש מתחיל"

עמוד שמייצר תעודה מוכנה לדפוס לכל נרשם: **כריכה** (עם הלוגו החדש ומכתב "משפחת סלקום") + **עמוד פנימי** עם השם, החוזקות, החלומות, הברכה והתמונה.

## איך משתמשים

1. **מוציאים את הנתונים מהגיליון:** ב-Google Sheets → `File → Download → Comma-separated values (.csv)`.
2. **פותחים** את `certificates/index.html` בדפדפן (דאבל-קליק).
3. **טוענים נתונים:** מדביקים את תוכן ה-CSV בתיבה ולוחצים "טען", או מעלים את קובץ ה-CSV.
4. **תמונות** (אחת מהאפשרויות):
   - מעלים את תיקיית/קובצי התמונות בכפתור "העלאת תמונות" — הן משובצות לפי שם הילד/ה (השם שבו נשמרו: `שם_משפחה`).
   - או: אם קישורי ה-Drive נמצאים בגיליון (עמודת "תמונה") — התמונות ייטענו אוטומטית (דורש חיבור לאינטרנט; הקבצים משותפים "כל מי שיש לו קישור").
5. **הדפסה / PDF:** לוחצים "🖨️ הדפס / שמור PDF" ובחלון ההדפסה:
   - יעד: **שמירה כ-PDF**
   - שוליים: **ללא (None)** (מומלץ)
   - הרקע והעיצוב נדפסים אוטומטית (התבנית היא תמונה) — לא חובה לסמן "גרפיקת רקע".
   - כל ילד = 2 עמודים (כריכה + פנים), A4 לרוחב.

> טיפ: להצצה מהירה בלי נתונים — פותחים `certificates/index.html#demo` (מוסיפים `#demo` לכתובת) ונטענת תעודת דוגמה.

כל ילד = 2 עמודים (כריכה + פנים). מומלץ להדפיס דו-צדדי.

## זיהוי עמודות
העמודות מזוהות אוטומטית לפי הכותרות בגיליון: `שם הילד/ה`, `שם משפחה`, `חוזקות`, `חלומות`, `ברכה אישית`, `תמונה`.
